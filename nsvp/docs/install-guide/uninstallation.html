Uninstalling the NexentaStor vCenter Plugin

This section demonstrates how to uninstall the NexentaStor vCenter Plugin, in Windows and Linux environments.

To uninstall the NexentaStor vCenter Plugin in a Windows environment, do the following:

  1. Run the Web Client Plugin .msi file
  2. Follow the prompts

To uninstall the NexentaStor vCenter Plugin in a Linux environment, enter the following command:

./install.sh -action uninstall -vcuser administrator@vsphere.local
